A. Card Grid

Repos as individual cards in a 3-column grid. Each card shows name, branch, status, ahead/behind, and action buttons. Good visual separation. Feels like a dashboard. Expandable detail would appear below the card.

B. Stacked Cards

Full-width cards stacked vertically. Two-line layout inside each card — name + badge + buttons on top, details underneath. Clean separation between repos. Expand/collapse detail drops in naturally below each card.
