State Behavior

  • Fetching/Pulling: Gray left border, spinning icon in pill, buttons disabled, slight opacity reduction
  • Error: Red left border + red pill with ✗, error message shown inline in monospace, "Retry" replaces Fetch button
  • Bulk progress: Header shows "Fetching 4/7…", Fetch All button disabled during operation
  • After operation: Card snaps back to its new status (green if clean, blue if behind, etc.)
