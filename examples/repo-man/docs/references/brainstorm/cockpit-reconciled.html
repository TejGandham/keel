Reconciled Design Decisions

  • Dark-first per CLAUDE.md, light as equal citizen via prefers-color-scheme
  • Flow grid, uniform cards — cards size to content, wrap naturally
  • Everything visible — dirty files, branches, pull-blocked reasons all inline. No expand/collapse.
  • Left border = status color — instant visual scan
  • Status pills for non-green states — fills for accessibility per CLAUDE.md
  • Monospace for git data (branches, file paths), system sans-serif for everything else
  • Action buttons only where relevant — clean repos don't show buttons
  • Linear/Vercel polish — subtle borders, rounded corners, good spacing
